# Security

This is a local-first research prototype. It does not intentionally contact
remote services, execute untrusted model output, or require API keys.

If you use it with agent traces, repository contents, or benchmark artifacts,
review those inputs before publishing them. They may contain secrets, private
code, prompts, or customer data.

Please report security issues privately by opening a minimal GitHub issue that
asks for a maintainer contact path, without disclosing exploit details.
